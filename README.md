# Проект: Место

### Обзор

Проект направлен на совершенствование навыков верстки, применения технологии grid, а также созданию модальных окон для ввода данных.

Точки "перелома" для "адаптивной верстки":

320 px
768 px
1024 px
1280 px

### Применяемые технологии

HTML 5, CSS4
